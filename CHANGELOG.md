# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-02-04

### Added
- Initial release
- Real-time working hours tracking in the top panel
- Visual progress bar with color-coded status
- History dialog with weekly and monthly views
- Customizable daily goal hours
- Remaining time display option
- Customizable status colors (in progress, completed, overtime)
- Multi-language support (English and French)
- Settings preferences panel
- Local data persistence in JSON format

### Features
- Support for GNOME Shell versions 42-47
- Automatic daily reset
- Quick history preview in dropdown menu
- Detailed history with navigation (previous/next week/month)
- Statistics showing total and average hours
